### Text Formatting

#### Headers
``` markdown
# is for <h1> tag
## is for <h2> tag and so on
```

#### Bold & Italic
``` markdown
**Text** and __Text__ will make it bold
*Text* and _Text_ will make it italic
```

#### Link & Images
``` markdown
[Text to Display](/image/path/here.jpg)
[Text to Display](link here)
```

#### Lists
``` markdown
Unordered lists
* Item 1
* Item 2
    * Item 2.1
    * Item 2.2

Ordered list
1. Item 1
1. Item 2
    1. Item 2.1
    1. Item 2.2
```
